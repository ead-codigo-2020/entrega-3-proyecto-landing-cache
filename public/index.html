¿Qué es la memoria caché?

Es un área de almacenamiento dedicada a la recuperación a gran velocidad de los datos usados o solicitados con más frecuencia que permite reutilizar de forma eficaz los datos procesados con anterioridad.

Reducción de latencia

Opera menor tiempo en acceder a los datos desde la memoria.

Mejor rendimiento del ancho de banda

Reduce el ancho de banda y aumenta la velocidad de descarga.

Almacenamiento de datos

Elimina los registros menos usados para hacer espacio a datos nuevos.

Tipos de caché

Caché de disco duro

Es una porción de memoria RAM asociada a un disco. Almacena datos recientemente leídos y agiliza su carga en el caso de que sean solicitados otra vez.

Caché de pista

Es una memoria de estado sólido, tipo RAM, dentro del disco duro. Utiliza memorias construidas con semiconductores para almacenar la información.

Caché web

Es la encargada del almacenamiento temporal de documentos web, como páginas web, imágenes y otros tipos de multimedia para reducir el ancho de banda consumido, la carga de los servidores y el retraso de las descargas. Un caché web almacena copias de los documentos que pasan por él, de forma que subsiguientes peticiones pueden ser respondidas por el propio caché, si se cumplen ciertas condiciones.
